# Typescript---Micromobilita-condivisa

DESCRIZIONE

Questo progetto modella in TypeScript il funzionamento di un servizio di micromobilità condivisa, ispirato a una piattaforma reale come Moove. 
L'obiettivo è rappresentare in modo chiaro e scalabile le interazioni tra:
- utenti
- mezzi di trasporto
- città


OBIETTIVO DEL PROGETTO

Il progetto è stato realizzato per:
- applicare i concetti di TypeScript avanzato
- modellare un dominio reale con classi e interfacce
- dimostrare una buona separazione delle responsabilità


TECNOLOGIE UTILIZZATE

- TypeScript
- Programmazione orientata agli oggetti (OOP)
- Interfacce e classi


STRUTTURA DEL PROGETTO

- 'interfaces.ts': definisce le interfacce del dominio (IMezzo, IUtente, ICitta).
- 'models.ts': implementa la logica principale tramite classi concrete (Mezzo, Utente, Citta).
- 'main.ts': contiene uno scenario di utilizzo e test del sistema


ESEMPIO DI UTILIZZO

- Creazione di una città
- Aggiuntà di mezzi (bici, scooter, monopattino)
- Prenotazione di un mezzo da parte di un utente
- Visualizzazione dello stato dei mezzi

CONCLUSIONI
Questo progetto mostra come TypeScript possa essere utilizzato per modellare sistemi reali in modo chiaro, tipizzato e manutenibile, ponendo le basi per applicazioni più complesse e scalabili.
